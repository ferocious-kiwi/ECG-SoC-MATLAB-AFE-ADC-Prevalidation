# Frequency Response Numerical Reference

CSV: `matlab_afe_validation/results_dataset/afe_frequency_response_metrics.csv`

본 결과는 XMODEL 구현 전 MATLAB reference frequency response이다. absolute magnitude는 IA gain ×201을 포함한 전체 MATLAB nominal chain gain이다. 따라서 0.05 Hz 같은 baseline-drift 영역에서도 absolute dB 값이 양수로 보일 수 있다. 해석에는 `relative_to_10Hz_passband_dB` 또는 `relative_to_passband_mean_dB` column을 함께 사용한다. 이 relative column은 ECG passband 기준 대비 baseline-drift/고주파/노치 주파수가 얼마나 상대적으로 감쇠되는지 보여주기 위한 보조 지표이다.

MATLAB time-domain chain의 60 Hz digital notch approximation은 정확히 60 Hz에서 ideal zero를 만들 수 있다. 따라서 60 Hz의 numerical dB 값은 physical analog attenuation claim이 아니며, CSV에서는 `< -120 dB` reporting cap으로 해석한다. 최종 논문에서 사용할 analog-style notch attenuation claim은 active Twin-T dense sweep 결과를 사용한다.
